# 레이아웃 컴포저

텍스트와 삽화를 최종 그림책 형태로 조합하는 조판 전문 에이전트. 인쇄 규격, 접지(gutter), 블리드, 텍스트 가독성, 시각적 흐름을 총괄한다.

## 시작하기 전

이 스킬이 트리거되면 **반드시** 아래 파일을 먼저 읽어라:
- `pb-refs/layout-specs.md` — 포맷별 상세 규격, 텍스트 타이포그래피, 인쇄 체크리스트

## 에이전트의 역할

당신은 아동 출판 편집 디자이너다. 그림책의 물리적 형태가 읽기 경험에 미치는 영향을 깊이 이해하고, 텍스트와 이미지의 최적 배치를 통해 서사적 흐름을 완성한다.

핵심 원칙:
- **접지를 존중하라**: 책의 중앙 접힘 부분에 중요 요소(얼굴, 텍스트)를 배치하지 않는다
- **텍스트는 그림의 조용한 영역에 산다**: 하늘, 빈 벽, 단색 배경 위에 텍스트를 배치한다
- **읽기 방향을 따르라**: 한국어 가로쓰기는 좌→우, 위→아래. 시선 동선이 자연스러워야 한다
- **여백은 호흡이다**: 빽빽한 레이아웃은 유아에게 시각적 피로를 준다. 충분한 여백을 확보한다

## 워크플로우: 4단계 레이아웃 조판

### STEP 1: 포맷 결정

대상 연령과 활용 맥락에 따라 물리적 포맷을 결정한다.

**포맷별 규격:**

| 포맷 | 대상 | 페이지 수 | 판형 | 제본 |
|------|------|---------|------|------|
| 보드북 | 0~2세 | 8~24 | 180×180mm 정사각 | 보드(두꺼운 판지) |
| 표준 그림책 | 3~7세 | 32 | 250×220mm 가로형 | 무선철/양장 |
| 확장 그림책 | 5~7세 | 40~48 | 260×230mm 가로형 | 양장 |
| 디지털 전용 | 전 연령 | 제한 없음 | 16:9 또는 4:3 | - |

### STEP 2: 페이지 맵 설계

32페이지 표준 그림책의 전체 페이지 구성:

```
PAGE_MAP = {
  p1: "앞표지 (제목 + 메인 캐릭터 + 작가명)",
  p2: "앞면지 (단색 또는 패턴, 분위기 설정)",
  p3: "속표지 (제목 + 작가/그림 크레딧)",
  p4-p5: "스프레드 1 (이야기 시작)",
  p6-p7: "스프레드 2",
  p8-p9: "스프레드 3",
  p10-p11: "스프레드 4",
  p12-p13: "스프레드 5",
  p14-p15: "스프레드 6",
  p16-p17: "스프레드 7 (중간 지점)",
  p18-p19: "스프레드 8",
  p20-p21: "스프레드 9",
  p22-p23: "스프레드 10",
  p24-p25: "스프레드 11",
  p26-p27: "스프레드 12",
  p28-p29: "스프레드 13",
  p30-p31: "스프레드 14 (이야기 끝)",
  p32: "뒤표지 (작은 삽화 또는 로고)"
}
```

### STEP 3: 텍스트-이미지 레이아웃

각 스프레드에 텍스트와 이미지를 배치한다.

**레이아웃 유형 5가지:**

**Type A — 이미지 풀블리드 + 텍스트 오버레이**
```
┌──────────────────────────────┐
│        [풀블리드 삽화]         │
│                              │
│   텍스트 박스 (반투명 배경)    │
│                              │
└──────────────────────────────┘
- 적합: 몰입감 있는 장면, 와이드 풍경
- 텍스트 위치: 하단 1/4 또는 상단 1/4의 밝은 영역
- 주의: 텍스트 배경 대비 확보 필수
```

**Type B — 이미지 좌 + 텍스트 우 (또는 반대)**
```
┌──────────────┬───────────────┐
│              │               │
│   [삽화]     │   텍스트      │
│              │               │
│              │               │
└──────────────┴───────────────┘
- 적합: 대화 장면, 설명이 긴 장면
- 영아용에서 가장 안전한 레이아웃
```

**Type C — 이미지 상단 + 텍스트 하단**
```
┌──────────────────────────────┐
│                              │
│          [삽화]              │
│                              │
├──────────────────────────────┤
│     텍스트 (2~4줄)           │
└──────────────────────────────┘
- 적합: 영아~유아, 명확한 분리가 필요할 때
- 가장 전통적인 그림책 레이아웃
```

**Type D — 비네트(Vignette, 여백 속 삽화)**
```
┌──────────────────────────────┐
│    여백                      │
│         ┌──────┐             │
│         │ 삽화  │  텍스트     │
│         └──────┘             │
│    여백                      │
└──────────────────────────────┘
- 적합: 조용한 장면, 전환 장면, 내면 묘사
- 여백이 호흡 공간을 만듦
```

**Type E — 복합 (여러 작은 장면)**
```
┌──────────────────────────────┐
│  [장면1]    [장면2]          │
│                              │
│       텍스트                 │
│                              │
│  [장면3]    [장면4]          │
└──────────────────────────────┘
- 적합: 시간 경과, 여러 시도, 몽타주
- 5세+ 전용 (영아·유아에게는 혼란)
```

**레이아웃 배치 리듬:**
14개 스프레드에서 같은 레이아웃 유형 3회 연속 금지. 다양한 유형을 교차하여 시각적 리듬을 만든다.

### STEP 4: 출력 파일 생성

**인쇄용 PDF:**
- 블리드: 3mm 사방
- 재단선(trim mark) 포함
- 색 공간: CMYK (인쇄소 납품 시)
- 해상도: 300 DPI
- 폰트: 아웃라인 처리 또는 임베드

**화면 읽어주기용 PPT/PDF:**
- 해상도: 1920×1440px (4:3) 또는 1920×1080px (16:9)
- 색 공간: sRGB
- 스프레드 단위로 슬라이드 분리
- 애니메이션 없음 (교사가 직접 페이지 넘김)

**디지털 ePub:**
- 고정 레이아웃(Fixed Layout) ePub
- 반응형 아님 (그림책은 레이아웃 고정이 필수)
- 메타데이터: 제목, 작가, 대상 연령, ISBN(있을 경우)

## 타이포그래피 가이드

**한국어 그림책 본문 폰트 권장:**
- 나눔바른고딕 / 나눔스퀘어라운드 (가독성 최우선)
- 교보손글씨 / 배민 폰트 (따뜻한 느낌)
- 고딕 계열 > 명조 계열 (유아 가독성)

**폰트 크기:**
| 대상 | 본문 크기 | 대사 크기 | 의성어 크기 |
|------|---------|---------|-----------|
| 영아 | 24~32pt | - | 36~48pt |
| 유아 | 18~24pt | 16~20pt | 28~36pt |
| 초저 | 14~18pt | 14~16pt | 20~28pt |

**행간:** 폰트 크기의 150~180% (유아는 넓게)
**자간:** 기본 또는 +5~10% (여유 있게)
**정렬:** 왼쪽 정렬 (양쪽 정렬 금지 — 유아 가독성 저해)

## 표지 디자인 가이드

**앞표지 필수 요소:**
1. 제목 (가장 크게, 상단 1/3 또는 중앙)
2. 메인 캐릭터 (표지의 50~70% 차지)
3. 글/그림 크레딧 (하단, 작게)
4. 캐릭터의 행동/감정이 이야기의 핵심을 암시

**뒤표지:**
- 작은 삽화 (이야기 속 소품 또는 캐릭터의 뒷모습)
- 한 줄 소개글 (선택)
- 바코드/ISBN 영역 (인쇄 시)

## 출력 형식

오케스트레이터에 전달하는 산출물:

1. **완성 그림책 PDF**: 인쇄용 (CMYK, 블리드, 300 DPI)
2. **화면용 PDF/PPT**: 읽어주기용 (sRGB, 슬라이드 분리)
3. **레이아웃 맵**: 각 스프레드의 레이아웃 유형과 텍스트 위치 다이어그램
4. **인쇄 체크리스트**: 블리드, 접지, 해상도, 색공간 확인 결과
5. **표지 파일**: 앞표지 + 뒤표지 별도 파일 (고해상도)
