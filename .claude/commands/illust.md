# /illust — 이수지 스타일 미니멀 일러스트 프롬프트 에이전트

> 한국 그림책 작가 이수지(Suzy Lee)의 미니멀리즘 미학을 기반으로,
> 나노바나나프로에 최적화된 일러스트 프롬프트를 생성합니다.

## 사용법

```
/illust <장면 설명>
/illust <용도> <장면 설명>
```

**용도** (선택): `커버` | `삽화` | `온보딩` | `마케팅` | `아이콘`
용도를 생략하면 범용 일러스트 프롬프트를 생성합니다.

**예시:**
- `/illust 아이가 꽃밭에서 책을 읽는 장면`
- `/illust 커버 엄마와 아이가 손잡고 걷는 모습`
- `/illust 삽화 숲속에서 여우를 만나는 아이`

---

## 핵심 미학: "여백의 미" 원칙

이수지 작가의 그림은 "그리지 않은 것"이 "그린 것"만큼 중요합니다.
이 스킬이 생성하는 모든 프롬프트는 아래 미학 원칙을 반영해야 합니다.

### 시각적 DNA (모든 프롬프트에 반영)

| 요소 | 규칙 | 이유 |
|------|------|------|
| **여백** | 화면의 80% 이상이 순백 또는 아주 옅은 크림색 배경 | 여백이 주인공을 더 돋보이게 함 |
| **선** | 섬세한 연필 또는 잉크 라인, 완벽하지 않은 손그림 느낌 | 따뜻하고 인간적인 질감 |
| **색채** | 1-2가지 은은한 수채화 포인트 컬러만 사용 | 절제된 아름다움, 과하면 촌스러움 |
| **밀도** | 피사체는 화면의 15-25%만 차지 | 나머지는 숨 쉴 공간 |
| **구도** | 피사체는 반드시 화면 중앙에 배치 | 1:1 크롭 시 캐릭터가 잘리지 않도록 |
| **분위기** | 고요하고 시적(poetic), 내면의 울림 | 치유적 동화의 정서와 일치 |
| **터치** | 수채화 번짐은 극히 절제, 연필 텍스처 위에 살짝 얹힌 정도 | 수채화가 주인공이 아닌 악센트 |

### 절대 피해야 할 것

- 짙은/어두운 배경, 꽉 찬 구도
- 선명한 채도 높은 색상 (디즈니/픽사 느낌)
- 복잡한 배경 디테일
- 사실적 렌더링이나 3D 느낌
- 애니메이션/만화 스타일 눈/표정
- 텍스트나 타이포그래피 요소

## 프롬프트 생성 프로세스

### 1단계: 장면 분석

사용자의 장면 설명에서 추출:
- **주인공**: 누가 등장하는가 (아이, 동물, 사물)
- **행위**: 무엇을 하고 있는가
- **환경**: 어디에 있는가 (최소한으로)
- **감정**: 어떤 정서를 전달하는가
- **포인트 컬러**: 장면에 어울리는 1-2가지 수채화 색상

### 2단계: 용도별 최적화

| 용도 | 비율 | 특별 고려사항 |
|------|------|-------------|
| **커버** | 1:1.4 (세로) | 하단 30%에 제목 공간 확보, 피사체는 상단-중앙에 배치 |
| **삽화** | 3:2 (가로) | 장면의 한 순간을 포착, 여백으로 상상의 공간 확보 |
| **온보딩** | 1:1 (정사각) | 중앙 집중 구도, 작은 사이즈에서도 읽히는 명확한 실루엣 |
| **마케팅** | 16:9 (가로) | 좌측에 피사체, 우측에 텍스트 배치 공간 |
| **아이콘** | 1:1 (정사각) | 극도로 단순화, 한 가지 요소만, 아이콘급 미니멀 |
| **범용** | 지정 없음 | 가로 또는 세로 중 장면에 맞는 방향 자동 선택 |

### 3단계: 프롬프트 조립

아래 구조로 영문 프롬프트를 조립합니다 (나노바나나프로는 영문 프롬프트가 최적):

```
[스타일 기반] + [장면 묘사] + [색채 지시] + [구도/여백 지시] + [분위기 지시]
```

**스타일 기반 (모든 프롬프트 공통 프리픽스):**
```
Minimalist children's book illustration in the style of Suzy Lee,
delicate pencil line drawing with barely-there watercolor accents,
vast white negative space occupying 80% of the composition,
```

**색채 지시 패턴:**
```
only [color1] and [color2] watercolor touches, extremely subtle and translucent,
the color should feel like a whisper not a shout,
```

**구도/여백 지시 패턴:**
```
the subject is centered in the composition for safe cropping in any aspect ratio,
the subject occupies only [15-25]% of the frame,
[용도별 구도 지시],
breathing room around every element,
```

**분위기 지시 패턴:**
```
quiet and poetic atmosphere, evoking a sense of [감정],
the kind of image that makes you pause and feel something,
clean bright airy feeling, no dark shadows
```

### 4단계: 3가지 변주 생성

같은 장면에서 3가지 다른 해석을 제공합니다:

| 변주 | 접근 | 설명 |
|------|------|------|
| **A. 클로즈업** | 피사체에 가까이 | 감정과 디테일 강조, 더 친밀한 느낌 |
| **B. 미디엄샷** | 적당한 거리 | 인물과 환경의 균형, 가장 안전한 선택 |
| **C. 와이드/시적** | 멀리서 바라봄 | 여백 극대화, 고독하지만 아름다운 분위기 |

## 품질 체크리스트

프롬프트 생성 후 자가 점검:

- [ ] 여백 80% 이상 확보 지시가 있는가?
- [ ] 색상이 2가지 이하인가?
- [ ] "subtle", "barely-there", "whisper" 등 절제 키워드가 있는가?
- [ ] 배경이 과도하게 묘사되지 않았는가?
- [ ] 피사체 크기가 15-25%로 제한되어 있는가?
- [ ] 피사체가 화면 중앙에 배치되어 1:1 크롭에도 안전한가?
- [ ] 어두운 톤이나 그림자 지시가 없는가?
- [ ] "clean", "bright", "airy" 등 밝음 키워드가 있는가?
- [ ] 나노바나나프로에서 잘 작동할 영문 프롬프트인가?

## 주의사항

- 프롬프트는 반드시 **영문**으로 출력 (나노바나나프로 최적화)
- 설명과 추천은 **한국어**로 출력
- "이수지 스타일"이라는 표현은 프롬프트 내에서 "in the style of Suzy Lee" 또는 "Suzy Lee inspired"로 사용
- 실제 이수지 작가의 특정 작품을 직접 참조하지 않음 (저작권 보호)
- 나노바나나프로 특성상 프롬프트가 길수록 좋으므로, 각 프롬프트는 3-5문장 분량으로 충분히 상세하게 작성
