# /deploy — 빌드 → 테스트 → 커밋 → 푸시 자동화

배포 전 전체 검증 파이프라인을 실행합니다.

## 순서

1. `npm run build` 실행하여 Edge Runtime 호환 빌드 확인
2. `npm test` 실행하여 전체 테스트 통과 확인
3. 빌드/테스트 실패 시 → 에러 내용 보고, 배포 중단
4. 성공 시 → `git status`로 변경사항 확인
5. 변경사항이 있으면 → 적절한 커밋 메시지로 커밋
6. `git push origin main`으로 Cloudflare Pages 배포 트리거
7. 최종 결과 보고

## 주의사항

- 빌드 실패 시 절대 push하지 마세요
- 커밋 메시지는 변경 내용을 정확히 반영해야 합니다
- `.env` 파일이 stage 되지 않았는지 반드시 확인하세요
