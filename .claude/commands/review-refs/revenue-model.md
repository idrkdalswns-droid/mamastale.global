# SaaS 수익 모델 분석 — mamastale 전용

## 핵심 SaaS 메트릭 정의

```
MRR (Monthly Recurring Revenue):
  mamastale은 구독이 아닌 건별 구매이므로:
  = 월 유료 전환 사용자 × 평균 객단가

CAC (Customer Acquisition Cost):
  = 마케팅 총비용 / 신규 유료 사용자 수
  0원 마케팅 시 CAC = 0 (오가닉만), 하지만 시간 비용 포함해야 현실적

LTV (Lifetime Value):
  = 평균 객단가 × 평균 구매 횟수
  건별 구매: LTV = 첫 구매 + (재구매율 × 평균 객단가 × 재구매 횟수)
  구독 전환 시: LTV = ARPU / 월 이탈률

LTV:CAC 비율:
  건강한 SaaS: 3:1 이상
  1:1 이하면 고객 획득할수록 손실

Payback Period:
  = CAC / (월 매출/고객)
  12개월 이하가 건강, 18개월+ 은 위험
```

## 100가지 시나리오 시뮬레이션

```
보수적 경로 (10%ile):
  월 500명 × 5% 가입 × 3% 유료 = 0.75명/월 유료
  연간 유료 고객 ~9명 × 객단가 ₩4,900 = 연 ₩44,100
  → 1년 매출: ~₩50,000 (5만원)

현실적 경로 (50%ile):
  월 2,000명 × 10% × 7% = 14명/월 유료
  연간 ~168명 × 평균 ₩8,000 = ₩1,344,000
  + 재구매 15%: ₩201,600
  → 1년 매출: ~₩1,500,000 (150만원)

낙관적 경로 (90%ile):
  월 8,000명 × 20% × 15% = 240명/월 유료
  연간 ~2,880명 × 평균 ₩14,000 = ₩40,320,000
  + 재구매 30%: ₩12,096,000
  → 1년 매출: ~₩52,000,000 (5,200만원)

🎯 1억 달성 시나리오:
  필요: 연 10,000명 유료 × 객단가 ₩10,000
  필요 트래픽: 월 119,000명 방문
  → 월 12만 방문은 SEO+커뮤니티+바이럴 없이는 불가능
  → 1억 달성 확률: 현재 계획으로는 ~5%
```

## Pre-mortem 분석

```
시나리오 1 (확률 35%): 트래픽 부족
  산후우울증 엄마는 "AI 동화 서비스"를 검색하지 않는다.
  발견 경로가 불명확. 병원·산후조리원 제휴 없이 자연 유입은 극히 적음.

시나리오 2 (확률 25%): 무료 → 유료 전환 실패
  무료 1편으로 충분하다고 느끼거나, 유료 품질 차이를 체감 못 함.
  "한 번 해봤는데 그냥 ChatGPT랑 비슷하네"라는 인식.

시나리오 3 (확률 20%): 리텐션 붕괴
  동화 1편 완성 후 재방문 이유 부재.
  "동화를 또 만들 이유"가 설계되지 않음 (성장 기록, 시리즈, 시즌).

시나리오 4 (확률 10%): 경쟁 진입
  GPT 기반 유사 서비스 등장. 또는 ChatGPT 자체에서 유사 경험 가능.
  진입 장벽(moat)이 약함.

시나리오 5 (확률 10%): 규제·신뢰 이슈
  AI 상담의 법적 리스크. 위기 상황 부적절 대응으로 언론 보도.
  개인 감정 데이터의 유출/오남용 우려.
```
