# 레이아웃 상세 규격 및 인쇄 체크리스트

## 인쇄 규격 상세

### 표준 그림책 (32페이지)

```
판형: 250mm(가로) × 220mm(세로)
스프레드: 500mm × 220mm (양면 펼침)
블리드: 3mm 사방
안전 영역: 재단선에서 안쪽 5mm
접지(Gutter): 중앙 좌우 각 8~10mm (총 16~20mm)

접지 안전 규칙:
- 접지 영역(중앙 ±10mm)에 텍스트 배치 금지
- 접지 영역에 캐릭터 얼굴 배치 금지
- 접지를 가로지르는 수평선/패턴은 허용 (배경 연속)
- 접지에서 캐릭터 몸통이 걸치는 것은 허용 (얼굴만 금지)
```

### 보드북 (8~24페이지)

```
판형: 180mm × 180mm (정사각) 또는 170mm × 170mm
페이지 두께: 2~3mm (두꺼운 판지)
모서리: 라운드 처리 (R5~R8) — 영아 안전
블리드: 3mm
접지: 보드북은 스프레드 개념 없음 (각 페이지 독립)

특수 고려:
- 각 페이지가 독립 장면 (스프레드 넘김 불가)
- 넘기기 쉽도록 탭/노치 고려
- 내구성: 물, 침에 강한 코팅 필요 (UV 코팅 또는 PP 코팅)
```

### 디지털 전용 규격

```
화면 읽어주기 (TV/프로젝터):
  해상도: 1920×1080 (16:9) 또는 1920×1440 (4:3)
  파일: PPT 또는 PDF
  
태블릿 읽기:
  해상도: 2048×1536 (iPad 4:3) 또는 2560×1600 (안드로이드)
  파일: ePub (Fixed Layout) 또는 PDF
  
키즈노트/블로그 공유:
  해상도: 1200×900 또는 1080×810
  파일: JPEG/PNG (개별 이미지)
```

## 텍스트 배치 상세 규칙

### 가독성 최우선 원칙

**대비(Contrast):**
- 밝은 배경 위 어두운 텍스트: 검정 또는 짙은 갈색 (#333333~#1A1A1A)
- 어두운 배경 위 밝은 텍스트: 흰색 (#FFFFFF) + 얇은 그림자 또는 반투명 배경 박스
- 최소 대비율: 4.5:1 (WCAG AA 기준)

**텍스트 배경 박스 (이미지 오버레이 시):**
```css
background: rgba(255, 255, 255, 0.75);  /* 반투명 흰색 */
padding: 12px 20px;
border-radius: 8px;  /* 부드러운 모서리 */
```
- 박스 색상은 해당 스프레드의 주조색과 조화
- 완전 불투명(1.0) 박스는 그림을 가리므로 지양

**텍스트 위치 결정 순서:**
1. 삽화에서 가장 밝거나 단순한 영역 찾기
2. 캐릭터 얼굴/행동과 겹치지 않는 곳 확인
3. 접지 근처 확인 (금지 영역)
4. 읽기 순서(좌→우, 위→아래) 자연스러운지 확인

### 의성어·의태어의 특수 타이포그래피

의성어와 의태어는 본문과 다른 시각적 처리를 한다:

```
"쿵!" → 크고 굵은 글씨, 약간의 기울임, 강조색
"살랑살랑" → 부드러운 곡선 폰트, 물결 배치
"타다다다" → 점점 커지는 글씨 크기
"아삭!" → 음식 색상(빨강/주황)의 글씨
```

**의성어 배치 원칙:**
- 소리의 발생원 근처에 배치 (발이 쿵! → 발 근처)
- 움직임 방향으로 글씨가 흐르듯 배치
- 크기로 소리의 강도를 표현

## 시각적 흐름 설계

### 좌→우 읽기 동선

```
스프레드에서의 이상적 시선 흐름:

[1. 좌측 페이지 상단] ─→ [2. 좌측 페이지 중앙/하단]
         │
         ▼
[3. 우측 페이지 상단] ─→ [4. 우측 페이지 중앙/하단]
         │
         ▼
      [페이지 턴]
```

**시선 유도 도구:**
- 캐릭터의 시선 방향 (오른쪽을 바라봄 = 다음으로 진행)
- 움직임 방향 (좌→우 = 전진, 우→좌 = 후퇴/회귀)
- 밝은 영역이 시선을 끔 (초점이 되어야 할 곳을 밝게)
- 대각선이 시선을 유도 (좌상→우하 = 자연스러운 흐름)

### 페이지 넘김 유도

마지막 시선이 우측 하단에 도달하면 자연스럽게 페이지를 넘기고 싶어진다.

**효과적 배치:**
- 텍스트의 마지막 줄이 우측 페이지 하단에 위치
- 캐릭터가 우측 페이지 밖(오른쪽)을 향해 걸어감/날아감
- 서스펜스 텍스트("그때...!") 후 빈 공간

## 인쇄 전 최종 체크리스트

```
□ 모든 이미지가 300 DPI 이상인가?
□ 블리드 3mm가 모든 페이지에 적용되었는가?
□ 재단선 내부 5mm 안전 영역에 텍스트가 있는가?
□ 접지 ±10mm에 얼굴/텍스트가 없는가?
□ 텍스트가 모두 아웃라인 처리(또는 폰트 임베드)되었는가?
□ 색 공간이 CMYK로 변환되었는가? (인쇄용)
□ 페이지 순서가 올바른가? (홀수=우측, 짝수=좌측)
□ 표지의 등폭(spine width)이 계산되었는가?
□ 바코드/ISBN 영역이 확보되었는가? (상업 출판 시)
□ 모서리 라운드 처리가 필요한가? (보드북)
□ 텍스트-배경 대비가 충분한가?
□ 의성어의 특수 타이포그래피가 인쇄에서도 선명한가?
```

## 표지 레이아웃 상세

### 앞표지 구성

```
┌─────────────────────────┐
│                         │
│      ★ 제목 ★           │  ← 상단 1/3, 가장 큰 폰트
│                         │
│    ┌─────────────┐      │
│    │             │      │
│    │  메인 캐릭터  │      │  ← 중앙, 표지의 50~70%
│    │  (행동 중)   │      │
│    │             │      │
│    └─────────────┘      │
│                         │
│  글 OOO · 그림 AI       │  ← 하단, 작은 폰트
└─────────────────────────┘
```

**표지 제목 폰트:**
- 본문보다 200~300% 큰 크기
- 둥근/친근한 서체 권장
- 손글씨 느낌의 폰트도 효과적
- 제목에 캐릭터와 연관된 색상 사용

### 등(Spine) 디자인

```
등폭 계산:
  32페이지 × 0.1mm(종이 두께) = 약 3.2mm (너무 얇아 텍스트 불가)
  양장 제본 시: +2mm 표지 두께 = 약 5.2mm (세로 제목 가능)
  
  보드북: 페이지당 2~3mm → 등폭 상당히 두꺼움
```

```
┌─────────────────────────┐

│   [작은 삽화]            │  ← 이야기 속 소품 또는

│                         │
│   "또리와 친구들의        │  ← 한 줄 소개 (선택)

│                         │
│   ┌──────┐              │
│   │바코드│              │  ← ISBN/바코드 영역 (인쇄 시)
│   └──────┘              │
└─────────────────────────┘
```
