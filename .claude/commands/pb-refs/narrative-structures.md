## 1. 연령-구조 매핑 테이블

| 구조 유형 | 영아(0~2) | 유아(3~4) | 유치원~초저(5~7) | 복잡도 |
|----------|:--------:|:--------:|:-------------:|:-----:|
| 반복     | ★★★     | ★★☆     | ★☆☆           | 최저   |
| 일상서사  | ★★★     | ★★☆     | ★☆☆           | 최저   |
| 누적     | ★☆☆     | ★★★     | ★★★           | 낮음   |
| 원형     | ☆☆☆     | ★★★     | ★★★           | 중간   |
| 문제-해결 | ☆☆☆     | ★★☆     | ★★★           | 중간~높음 |
| 점층     | ☆☆☆     | ★★☆     | ★★★           | 높음   |

★★★ = 최적, ★★☆ = 적합, ★☆☆ = 가능하지만 단순화 필요, ☆☆☆ = 부적합

## 2. 반복 구조 (Repetitive Structure)

**정의:** 동일한 문장/장면 패턴이 대상만 바꿔가며 반복되는 구조.
**적정 연령:** 0~4세 (영아기에 가장 효과적)
**대표작:** 《갈색 곰아, 갈색 곰아, 무엇을 보니?》(에릭 칼), 《사과가 쿵!》

**구조 공식:**
```
[도입 문장] + [대상 A] + [반복 패턴]
→ [동일 문장] + [대상 B] + [반복 패턴]
→ [동일 문장] + [대상 C] + [반복 패턴]
→ ... (5~8회 반복)
→ [변주/종결 문장]
```

**32페이지 배분:**
- 스프레드 1: 첫 번째 대상 + 패턴 확립
- 스프레드 2~11: 반복 (각 스프레드 1개 대상, 총 10개)
- 스프레드 12~13: 마지막 반복 + 변주 암시
- 스프레드 14: 전체 대상 총출동 또는 반전 종결

**설계 핵심:**
- 반복 패턴의 리듬감이 생명. 음절 수와 억양이 일정해야 아이가 따라할 수 있다
- 대상은 아이의 친숙도 순서로 배열 (가장 친숙한 것부터)
- 마지막 2~3회에서 미세한 변주를 넣어 지루함 방지
- 종결부에서 "예측 파괴" 1회가 효과적 (아이들이 가장 좋아하는 순간)

**영아(0~2)를 위한 조정:**
- 반복 횟수 축소: 5~6회
- 페이지당 단어: 1~5개
- 대상: 일상 사물/동물만 (추상 개념 불가)

## 3. 누적 구조 (Cumulative Structure)

**정의:** 새 사건이 추가될 때마다 이전 사건이 모두 반복되며 눈덩이처럼 커지는 구조.
**적정 연령:** 2~7세
**대표작:** 《잠자는 집》(오드리 우드), 《커다란 순무》, 《이 작은 아기 오리가》

**구조 공식:**
```
A
A → B
A → B → C
A → B → C → D
...
A → B → C → D → E → [절정/전복]
```

**32페이지 배분:**
- 스프레드 1~2: A 소개 + B 추가
- 스프레드 3~4: A+B 반복 + C 추가
- 스프레드 5~7: 누적 가속 (D, E, F 추가)
- 스프레드 8~10: 누적 최대치 (전체 나열이 한 스프레드를 가득 채움)
- 스프레드 11~12: 절정 (탑이 무너지거나 / 마지막 존재가 모든 것을 바꿈)
- 스프레드 13~14: 해소 + 종결

**설계 핵심:**
- 누적 요소의 수: 5~7개가 최적 (3세: 4~5개, 5세+: 6~7개)
- 각 요소는 이전보다 약간씩 특이하거나 크거나 웃긴 것이어야 흥미 유지
- 전체 누적 나열 시 리듬감 있는 나열 문장 필요 (운율/음절수 맞추기)
- 절정은 가장 작거나 약한 존재가 가장 큰 변화를 일으키는 반전이 효과적

## 4. 원형 구조 (Circular Structure)

**정의:** 이야기가 출발점으로 돌아오는 구조. 시작과 끝이 같은 장소/상황이지만 주인공의 내면이 변화.
**적정 연령:** 3~7세
**대표작:** 《쥐에게 쿠키를 주면》(로라 뉴메러프), 《우리 엄마》(앤서니 브라운)

**구조 공식:**
```
A(출발) → B → C → D → ... → A'(귀환, 하지만 변화된 상태)
```

**32페이지 배분:**
- 스프레드 1~2: 출발점 확립 (장소, 상황, 캐릭터의 상태)
- 스프레드 3~5: 여정/탐험의 시작과 전개
- 스프레드 6~10: 중간 여정 (발견, 만남, 경험)
- 스프레드 11~12: 전환점 (돌아가고 싶은 이유 발생)
- 스프레드 13: 귀환
- 스프레드 14: 출발점과 같은 장면이지만 미묘한 차이 (주인공의 변화 시각화)

**설계 핵심:**
- 마지막 스프레드가 첫 스프레드와 시각적으로 대응해야 함 (같은 구도, 다른 표정/색채)
- 인과관계 사슬이 자연스러워야 함 (A→B의 이유가 명확)
- 3~4세용은 물리적 원형 (같은 장소로 돌아옴), 5~7세용은 심리적 원형 (같은 상황이지만 다른 대응)

## 5. 문제-해결 구조 (Problem-Solution Structure)

**정의:** 주인공이 문제에 직면하고, 여러 시도 끝에 스스로 해결하는 구조. 유아용 그림책의 가장 보편적 서사.
**적정 연령:** 3~8세 (유아기 핵심 구조)
**대표작:** 《으뜸 헤엄이》(레오 리오니), 《무지개 물고기》(마르쿠스 피스터)

**구조 공식 (3의 법칙 적용):**
```
설정 → 문제 발생 → 시도1(실패) → 시도2(실패) → 깨달음 → 시도3(성공) → 새 균형
```

**32페이지 배분 (Bine-Stock 프레임워크):**

**1막 — 설정 (스프레드 1~3, 20%)**
- 스프레드 1: 세계관 + 캐릭터 소개 (평화로운 일상)
- 스프레드 2: 캐릭터의 욕구(desire) 또는 특성 드러남
- 스프레드 3: 문제 발생 (Inciting Incident) → **첫 번째 페이지 턴 훅**

**2막 — 시도와 실패 (스프레드 4~11, 60%)**
- 스프레드 4~5: 시도 1 — 가장 단순하고 직관적인 해결 시도 → 실패
  - 실패 이유: 문제를 잘못 이해했거나, 혼자만의 방법으로 접근
- 스프레드 6~8: 시도 2 — 다른 방법 시도, 누군가의 조언 참고 → 더 큰 실패
  - 실패 이유: 방향은 맞지만 핵심을 놓침
  - 여기서 감정적 최저점(emotional low) 배치
- 스프레드 9~10: 깨달음의 순간 (Epiphany)
  - 작은 사건이나 타인의 한마디가 핵심 통찰을 제공
  - 이 깨달음은 교육 목표와 직결되어야 함
- 스프레드 11: 시도 3 준비 — 새로운 관점으로 행동 시작 → **긴장감 최고조**

**3막 — 해결과 새 균형 (스프레드 12~14, 20%)**
- 스프레드 12: 시도 3 성공 — 깨달음을 적용하여 문제 해결
- 스프레드 13: 성공의 결과 — 주변의 반응, 기쁨의 공유
- 스프레드 14: 새로운 균형 — 변화된 주인공의 일상 (첫 스프레드와 대비)

**설계 핵심:**
- 3번의 시도는 반드시 점층적(escalating)이어야 함. 같은 수준의 시도 반복 금지
- 각 실패는 다음 시도를 위한 학습을 포함해야 함 (순수한 실패 금지)
- 깨달음은 외부에서 주입되는 것이 아니라, 주인공 내부에서 발생해야 함
- 시도3의 성공은 1+2의 경험이 축적된 결과여야 함 (돌연 성공 금지)
- **절대 규칙: 어른(부모/교사/마법사)이 대신 해결하는 결말 금지**

## 6. 점층 구조 (Escalation Structure)

**정의:** 사건의 규모나 강도가 점점 커지다가 한계점에서 폭발/전복되는 구조.
**적정 연령:** 4~7세
**대표작:** 《방귀쟁이 며느리》, 《모자 사세요》, 《까막나라에서 온 삽사리》

**구조 공식:**
```
작은 사건 → 좀 더 큰 사건 → 훨씬 큰 사건 → 감당 불가 → 폭발/전복 → 원상복귀 + 교훈
```

**설계 핵심:**
- 유머에 가장 적합한 구조 (점점 커지는 과장이 웃음을 유발)
- 각 단계의 크기 증가가 시각적으로 명확해야 함 (그림에서 크기가 실제로 커짐)
- 폭발/전복 순간이 "가장 재미있는 장면"이 되어야 함
- 원상복귀 후 작은 여운(또 시작될 것 같은 암시)이 효과적

## 7. 일상 서사 구조 (Slice-of-Life)

**정의:** 극적 갈등 없이 일상의 한 순간을 감각적으로 포착하는 구조.
**적정 연령:** 0~3세 (영아 최적)
**대표작:** 《달님 안녕》(하야시 아키코), 《잘 자요 달님》(마거릿 와이즈 브라운)

**구조 공식:**
```
일상의 시작 → 감각적 장면 1 → 감각적 장면 2 → ... → 일상의 마무리
```

**설계 핵심:**
- 갈등이 아닌 감각적 경험과 정서적 안정감이 중심
- 시간 흐름 (아침→저녁, 식사→잠자리) 이 자연스러운 구조를 제공
- 매 스프레드마다 하나의 감각(시각/청각/촉각)에 집중
- 반복적인 인사("안녕", "잘 자") 가 리듬을 만듦

## 8. 32페이지 범용 배분 템플릿

어떤 구조를 선택하든 활용 가능한 기본 비트(beat) 배분:

| 스프레드 | 비트 | 기능 | 감정 강도 |
|---------|------|------|:--------:|
| 1 | HOOK | 첫 이미지로 세계관 즉각 전달 | ●●○○○ |
| 2 | SETUP | 캐릭터와 일상 소개 | ●●○○○ |
| 3 | CATALYST | 변화의 시작/문제 발생 | ●●●○○ |
| 4 | RISING_1 | 첫 반응/시도 | ●●●○○ |
| 5 | RISING_2 | 상황 전개 | ●●●●○ |
| 6 | MIDPOINT | 중간 전환점 | ●●●●○ |
| 7 | RISING_3 | 긴장 고조 | ●●●●● |
| 8 | COMPLICATION | 예상치 못한 장애 | ●●●●● |
| 9 | DARKEST_MOMENT | 가장 어려운 순간 | ●●●●● |
| 10 | EPIPHANY | 깨달음/전환 | ●●●○○ |
| 11 | CLIMAX_PREP | 최종 시도 준비 | ●●●●○ |
| 12 | CLIMAX | 절정/해결 | ●●●●● |
| 13 | RESOLUTION | 결과와 반응 | ●●●○○ |
| 14 | NEW_NORMAL | 변화된 일상/여운 | ●●○○○ |

## 페이지 턴 유인(Page-Turn Hook) 패턴

**1. 질문형**: "그런데... 또리는 어떻게 했을까요?"
**2. 서스펜스형**: "바로 그때, 뒤에서 커다란 소리가!" (텍스트만, 그림은 다음 페이지)
**3. 시선 유도형**: 캐릭터가 오른쪽 페이지 밖을 바라보는 그림
**4. 동작 중단형**: 캐릭터가 행동 도중 멈추고 무언가를 발견
**5. 감정 변화형**: 캐릭터 표정만 클로즈업, 무슨 일인지는 다음 페이지
**6. 대비형**: 왼쪽 페이지의 기대와 완전히 다른 것이 오른쪽에 등장
