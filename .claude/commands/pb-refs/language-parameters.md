## 1. 연령별 어휘 수준 상세

### 영아(0~2세) 허용 어휘

**명사 (일상 사물):** 엄마, 아빠, 아기, 멍멍이, 야옹이, 공, 물, 밥, 까까(과자), 맘마(밥), 빠빠(자동차), 달, 해, 별, 꽃, 나무, 눈(snow), 비, 신발, 모자, 양말
**동사 (기본 행위):** 먹다, 자다, 울다, 웃다, 가다, 오다, 보다, 안다(hug), 주다, 앉다, 일어나다
**형용사 (기본 감각):** 크다, 작다, 예쁘다, 뜨겁다, 차갑다, 맛있다
**인사/반응:** 안녕, 까꿍, 짝짝짝, 잘 자, 사랑해

### 유아(3~4세) 추가 어휘

**감정 어휘:** 기쁘다, 슬프다, 화나다, 무섭다, 좋아하다, 싫어하다, 부끄럽다, 외롭다
**관계 어휘:** 친구, 우리, 함께, 같이, 나누다, 도와주다, 미안해, 고마워
**동작 어휘:** 달리다, 뛰다, 숨다, 찾다, 만들다, 그리다, 놀다, 던지다, 잡다
**자연 어휘:** 하늘, 바다, 산, 강, 숲, 동물 이름(토끼, 곰, 새, 물고기, 나비)
**크기·양:** 많다, 적다, 높다, 낮다, 길다, 짧다, 무겁다, 가볍다

### 초저(5~7세) 추가 어휘

**복합 감정:** 자랑스럽다, 설레다, 서운하다, 안타깝다, 뿌듯하다
**추상 개념:** 용기, 약속, 비밀, 모험, 꿈, 소원, 기회
**비유적 표현:** ~처럼, ~만큼, 마치 ~같은
**시간 개념:** 옛날옛날, 어느 날, 그 순간, 드디어, 갑자기
**논리 접속:** 그래서, 하지만, 그런데, 왜냐하면, 만약에

### 연령별 금지 어휘

**영아:** 추상 명사, 복합 감정, 비유, 3음절 이상 동사
**유아:** 학술 용어, 은유적 표현, 이중 부정, 가정법
**초저:** 성인 전용 관용구, 전문 용어 (단, 맥락 설명 동반 시 일부 허용)

## 2. 문장 구조 가이드

### 영아(0~2세)

```
허용 구조:
"멍멍!"                          (의성어 단독)
"큰 공!"                         (형용사+명사)
"엄마가 안아줘요."                 (주어+동사)
"까꿍! 여기 있지!"                (의성어+짧은 문장)

금지 구조:
"엄마가 아기를 안아주었습니다."     (과거형+존칭)
"빨간 공이 데굴데굴 굴러갔어요."   (너무 긴 문장)
```

### 유아(3~4세)

```
허용 구조:
"또리는 열매를 찾았어요."                    (주어+목적어+동사)
"'내 거야!' 또리가 소리쳤어요."              (대사+서술)
"뭉치는 슬펐어요. 눈물이 뚝."               (감정 서술+의태어)
"또리가 달려가요. 타다다다!"                 (행동+의성어)
"열매를 나눠 먹으니까 더 맛있어요."           (인과 복문, 제한적)

금지 구조:
"또리는 열매를 독차지하려다가 친구들이 슬퍼하는 것을 보고..." (중첩 관형절)
"만약 또리가 처음부터 나눴더라면..."           (가정법)
```

### 초저(5~7세)

```
허용 구조:
모든 유아 구조 + 
"또리는 처음에는 무서웠지만, 용기를 냈어요."    (대조 복문)
"뭉치가 '괜찮아' 하고 말했지만, 눈물이 흘렀어요." (행동+감정 대비)
"열매가 얼마나 맛있었는지, 모두 눈이 동그래졌어요." (정도 부사절)
"그날부터 또리는 달라졌어요."                   (시간 표현+변화)
```

## 4. 운율 설계 가이드

### 한국어 리듬의 기본 단위

한국어 그림책의 리듬은 **음절 수 맞추기**와 **끝소리(종성) 반복**으로 만든다.

**음절 수 맞추기:**
```
"뚜벅 뚜벅 걸어가요" (3+3+4 = 10음절)
"아장 아장 걸어가요" (3+3+4 = 10음절)
→ 같은 음절 패턴의 반복이 안정적 리듬 생성
```

**끝소리 반복 (유사 각운):**
```
"또리가 달려갔어요"
"뭉치가 따라갔어요"
"하하가 날아갔어요"
→ "~갔어요" 반복으로 리듬 완성
```

**3-3-4 패턴 (가장 안정적):**
한국어에서 가장 자연스러운 리듬 단위. 동요의 대부분이 이 패턴.
```
"빨간 열매 / 맛있겠다" (4+5)
"내 거야 / 내 거야 / 내 거야" (3+3+3)
"같이 먹자 / 같이 먹자" (4+4)
```

### 반복-변주 패턴

```
반복 1회: "열매를 하나 먹었어요. 냠!"
반복 2회: "열매를 둘 먹었어요. 냠냠!"
반복 3회: "열매를 셋 먹었어요. 냠냠냠!"
변주(반전): "그런데... 열매가 하나도 안 남았어요."
→ 수의 증가 + 소리의 증가 + 마지막 반전
```

## 5. 참여 유도 장치

### 유형별 참여 포인트

**1. 반복 합창형 (2세+):**
교사가 시작하고 아이들이 함께 따라하는 반복구
```
교사: "또리가 뭐라고 했을까?"
아이들: "내 거야!"
```

**2. 질문형 (3세+):**
아이들에게 다음 사건을 예측하게 하는 질문
```
"뭉치는 어떻게 했을까요?"
"너라면 어떻게 할 것 같아?"
```

**3. 동작형 (2세+):**
텍스트에 따라 몸을 움직이는 동작 유도
```
"또리가 기지개를 켰어요. (우리도 같이 기지개!)"
"쿵쿵쿵 발을 구르며 (발 구르기)"
```

**4. 소리형 (전 연령):**
의성어를 함께 내는 참여
```
"또리가 열매를 베어 물었어요. 아삭! (아삭!)"
```

**5. 카운팅형 (3세+):**
숫자를 함께 세는 참여
```
"열매가 하나, 둘, 셋... 다섯 개나 있어요!"
```

### 참여 포인트 배치 원칙

- 전체 14 스프레드 중 최소 3개, 최대 7개
- 첫 참여 포인트는 스프레드 2~3에 배치 (일찍 참여 패턴 확립)
- 절정(클라이맥스) 직전에 가장 강력한 참여 포인트 배치
- 같은 유형 연속 3회 이상 금지 (다양하게 섞기)
