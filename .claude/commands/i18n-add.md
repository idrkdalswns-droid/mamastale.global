# /i18n-add — 새 번역 키 6개 언어 동시 추가

새로운 번역 키를 6개 언어(ko, en, ja, zh, ar, fr) 파일에 동시 추가합니다.

## 사용법

`/i18n-add` 뒤에 추가할 키와 한국어 텍스트를 지정하세요.
예: `/i18n-add Chat.welcomeBack 다시 오셨군요!`

## 순서

1. 사용자가 지정한 키 이름과 한국어 텍스트 확인
2. `messages/ko.json`에 해당 키 추가 (한국어 원문)
3. 나머지 5개 언어 파일에 적절한 번역 추가:
   - `messages/en.json` — 영어
   - `messages/ja.json` — 일본어
   - `messages/zh.json` — 중국어(간체)
   - `messages/ar.json` — 아랍어
   - `messages/fr.json` — 프랑스어
4. 번역 시 기존 파일의 톤앤매너를 참고하여 일관성 유지
5. JSON 형식 검증 (파싱 오류 없는지 확인)
6. 결과 요약: 추가된 키, 6개 언어 번역 내용

## 주의사항

- 이미 존재하는 키는 덮어쓰지 않고 알림
- 한국어 톤: 따뜻하고 부드러운 존댓말
- 아랍어: RTL 호환 확인
- 일본어: 경어체 사용
