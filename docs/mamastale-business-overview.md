# MAMASTALE (엄마엄마동화) — 서비스 전체 개요

> 하노이/다낭 사업 제안용 원본 자료 (2026.03)
> 별도 에이전트에서 문서화 작업 진행 예정

---

## 1. 서비스 한 줄 정의

**어머니의 감정적 상처를 AI 4단계 치유적 대화를 통해 아이에게 읽어줄 10장면 동화로 변환하는 B2C SaaS**

- 핵심 약속: 엄마의 고통이 아이가 가장 좋아하는 잠자리 동화가 된다
- 소요 시간: 15-20분 대화 → 10장면 동화 1편 완성
- 타겟: 산후우울증, 양육 번아웃, 정체성 위기를 겪는 한국 어머니 (25-45세)
- 지원 언어: 한국어(기본), 영어, 일본어, 중국어, 아랍어, 프랑스어

---

## 2. 대표의 아이템 개발 역사

### 창업 동기 — 대표 강민준의 실제 경험

동유럽 자전거 여행 중 극심한 외로움을 경험. 매일 밤 노란 텐트 안에서 일기를 쓰기 시작. 글로 감정을 풀어내는 과정에서 치유를 경험.

"내가 경험한 이 치유적 글쓰기를, 아이를 키우며 지쳐있는 어머니들에게 줄 수 없을까?"

→ 페니베이커의 표현적 글쓰기 이론(40년 연구)을 기반으로, AI와의 대화를 통해 감정을 풀어내고, 그 결과물이 아이에게 읽어줄 동화가 되는 서비스 설계.

### 투자자 데모 동화: "노란 텐트 속 무지개 마법"

대표의 실제 이야기를 10장면으로 재구성한 프로토타입 동화. 서비스 설계의 출발점이자 증명.

## 3. 4단계 AI 치유 엔진 (핵심 기술)

### 설계 철학

4명의 AI 상담사가 순서대로 어머니와 대화. 각 단계는 독립적 임상 심리학 원리에 기반.

### Phase 1: 하은이 — 공감적 치유자 (감정 표현)

- 아이콘: 🫧 | 색상: #7FBFB0 (민트)
- 이론: 페니베이커의 능동적 억제 가설 + 리버만의 감정 명명 효과
- 역할: 판단 없는 무조건적 수용. 감정을 말로 표현하게 유도.
- 금지: 격려("힘내세요!"), 거짓 희망("다 괜찮을 거예요"), 일반화("다들 그래요")
- 규칙: 3-5문장 응답, 질문 1개씩, 최대 7턴
- 전환 신호: 어머니가 감정(feel) → 인지(think)로 전환될 때

### Phase 2: 민서 — 소크라테스식 철학자 (통찰 발견)

- 아이콘: 🌿 | 색상: #E07A5F (코럴)
- 이론: CBT 인지행동치료의 소크라테스식 질문법
- 역할: 숨겨진 강점 발견. 인지 왜곡 해체. Story Seed 추출.
- 기법: 증거 확인("나쁜 엄마라는 증거가 뭔가요?"), 탈중심화("친구에게라면 뭐라고 할 건가요?"), 예외 찾기("이 문제가 지배하지 못했던 순간은?")
- Story Seed 예시: 무조건적 사랑, 끈기, 연대, 용기
- 규칙: 질문 1개/응답, 최대 10턴

### Phase 3: 지우 — 은유의 마법사 (문제 외재화)

- 아이콘: ✨ | 색상: #8B6AAF (보라)
- 이론: 마이클 화이트의 내러티브 치료 — "사람이 문제가 아니라, 문제가 문제다"
- 역할: 어머니의 고통을 동화 캐릭터(괴물/마법 도구)로 의인화
- 14개 은유 매핑 테이블:

| 감정적 고통 | 동화 캐릭터 | 대항 주체 (마법 도구) |
|------------|-----------|-------------------|
| 산후우울증 | 마을의 짙은 안개 괴물 | 안개를 헤치는 횃불 |
| 양육 번아웃 | 불 뿜는 아기 용 | 용을 진정시키는 마법의 노래 |
| 경력 단절 | 길 잃은 꼬마 여우 | 여우를 안내하는 별 |
| 시댁 갈등 | 얼음 성에 갇힌 공주 | 얼음을 녹이는 햇빛 같은 말 |
| 자존감 상실 | 거울에 비치지 않는 요정 | 진실의 거울 |
| 죄책감/수치심 | 어깨 위의 무거운 돌맹이 요괴 | 요괴를 떨쳐내는 춤 |
| 화병/억눌린 분노 | 가슴 속 꺼지지 않는 작은 화산 | 화산을 식혀주는 맑은 샘물 |
| 완벽주의 | 금이 갈까 두려운 유리 인형 | 금 간 곳에서 빛이 새어나오는 금수선 |
| 외로움 | 사람들 속의 보이지 않는 벽 | 벽을 무너뜨리는 손 |
| 불안 | 밤마다 나타나는 검은 새 | 새를 몰아내는 옅은 빛 |

- 규칙: 2-3개 은유 제시 → 어머니 선택. 최대 10턴.
- v1.22.0 추가: 동화 생성 전 커스텀 질문 2개 (배경 + 특별 요소)

### Phase 4: 서연 — 동화 편집장 (10장면 서사 생성)

- 아이콘: 📖 | 색상: #C4956A (브라운)
- 이론: 루이스 디살보의 치유 서사 5대 요건 + 조셉 캠벨의 영웅의 여정
- 역할: 모든 단계의 결과물을 10장면 기승전결 동화로 완성

10장면 구조:

| 장면 | 구분 | 영웅의 여정 | 임상 목표 |
|------|------|-----------|-----------|
| 1-2 | 기 (도입) | 일상의 세계 | 문제 이전의 안전한 모성 사랑 확인 |
| 3-4 | 승 (갈등) | 모험의 소명 | 고통의 은유적 재현 (괴물 침입) |
| 5-6 | 전 (시도) | 시련과 동맹 | 숨겨진 강점 + 도움 발견 |
| 7-8 | 전 (해결) | 보상 | Story Seed가 마법 도구로 괴물 격퇴 |
| 9-10 | 결 (교훈) | 귀환 | 지혜를 아이에게 전승 (원형 구조) |

- 문체: 3-5세 아이도 이해하면서 어른이 읽어도 가슴이 울리는 수준
- 감각적 은유: "슬펐어요" (X) → "마음에 차가운 비가 내렸어요" (O)
- 자녀 연령별 조절: 0-2세(의성어/의태어), 3-5세(은유적 표현), 6-8세(내면 묘사)
- v1.22.0 추가: 10장면 일괄 생성, [STORY_END] 마커, AI 자동 제목 생성, 대사 줄바꿈

## 4. 위기감지 시스템 (CSSRS 기반)

### 3단계 프로토콜

| 수준 | 정의 | 대응 |
|------|------|------|
| LOW (Level 1) | 소극적 소망 | 로그만 기록 |
| MEDIUM (Level 2-3) | 비특이적 관념 | AI 프롬프트에 위기 맥락 주입, 부드러운 안전 확인 |
| HIGH (Level 4-5) | 구체적 의도/계획 | Claude API 바이패스, 하드코딩 위기 응답 즉시 반환 |

### HIGH 수준 응답 (AI 없이 하드코딩)

"지금 많이 힘드시군요. 어머니의 고통이 느껴집니다.
지금 이 순간, 어머니의 안전이 가장 중요합니다.

📞 자살예방상담전화: 1393 (24시간, 무료)
📞 정신건강위기상담전화: 1577-0199 (24시간)
📞 응급: 119"

### 오탐 방지 (14개 관용 표현 패턴)

"죽을 맛!" (맛있다), "웃겨죽겠다" (웃기다), "살인적인 일정" (바쁘다) 등 한국어 관용 표현 필터링

### 6개 언어 키워드 모니터링

한국어, 영어, 일본어, 중국어, 아랍어, 프랑스어 위기 키워드 실시간 감지

## 8. 선생님 모드 (B2B2C 확장)

### 타겟

유치원/어린이집 교사 (기관 코드 기반 접근)

### 5단계 가이드 동화 창작

| 단계 | 이름 | 설명 |
|------|------|------|
| Phase A | 씨앗 심기 | 주제 탐색, 상황 파악 |

## 11. 데이터 모델 (Supabase PostgreSQL)

### 핵심 테이블

| 테이블 | 용도 |
|--------|------|
| profiles | 사용자 프로필 (이름, 로케일, 잔여 티켓) |
| sessions | 채팅 세션 (현재 Phase, 상태) |
| messages | 대화 기록 (역할, 내용, Phase) |
| stories | 완성 동화 (제목, 장면 JSONB, 공개 여부) |
| feedback | 5차원 평가 (공감/통찰/은유/서사/전체) |
| subscriptions | Stripe 동기화 |
| community_likes | 좋아요 |
| community_comments | 댓글 |
| referrals | 추천 코드 |
| llm_call_logs | AI 호출 추적 (모델, 토큰, 비용, 지연) |
| crisis_events | 위기 이벤트 추적 (심각도, 키워드, 응답) |

### 보안

- 모든 테이블 RLS (Row Level Security) 활성화
- 사용자는 자신의 데이터만 접근 가능
- PKCE 인증 (OAuth)
- 원자적 티켓 차감 (경쟁 조건 방지)
