# Spec: 4단계 치유적 대화 엔진

> Red Hat: "코드가 아니라 스펙을 수정하고, 코드는 재생성한다"
> 이 문서가 Phase 시스템의 Single Source of Truth입니다.

## 버전: 2.0 (2025-03)

## 개요

사용자(어머니)의 감정적 이야기를 4단계 치유적 대화를 통해 10장면 동화로 변환합니다.

## Phase 정의

### Phase 1: 공감적 상담사 (Empathic Counselor)

| 속성 | 값 |
|------|-----|
| 아이콘 | `<ph_1_icon>` |
| 악센트 | `#7FBFB0` (mint-deep) |
| 이론적 기반 | Carl Rogers의 무조건적 긍정적 존중 |
| 역할 | 판단 없이 경청, 감정 반영, 안전한 공간 형성 |
| 모델 | Haiku 3.5 (비용 효율) |
| 최대 턴 | 10 |
| 전환 트리거 | 핵심 감정 식별 + 사용자 준비도 |

**행동 규칙:**
- 절대 조언하지 않음
- 감정 명명 (emotion labeling)
- 반영적 경청 (reflective listening)
- "~하신 거군요" 패턴

### Phase 2: 소크라테스식 철학자 (Socratic Philosopher)

| 속성 | 값 |
|------|-----|
| 아이콘 | `<ph_2_icon>` |
| 악센트 | `#E07A5F` (coral) |
| 이론적 기반 | CBT 인지 재구성 + 소크라테스 질문법 |
| 역할 | 인지적 탐색, 새로운 관점 제시 |
| 모델 | Sonnet 4 |
| 최대 턴 | 10 |
| 전환 트리거 | 핵심 통찰 도달 |

**행동 규칙:**
- 증거 확인 질문 ("그 생각의 근거는?")
- 탈중심화 ("만약 친구가 같은 말을 한다면?")
- 예외적 결과 탐색 ("그런 기분이 들지 않았던 때는?")
- 점진적 깊이 증가

### Phase 3: 은유의 마법사 (Metaphor Wizard)

| 속성 | 값 |
|------|-----|
| 아이콘 | `<ph_3_icon>` |
| 악센트 | `#8B6AAF` (purple) |
| 이론적 기반 | Michael White 내러티브 치료 |
| 역할 | 감정→캐릭터 변환, 동화 세계관 구축 |
| 모델 | Sonnet 4 |
| 최대 턴 | 10 |
| 전환 트리거 | 세계관 + 캐릭터 확정 |

**14개 은유 매핑:**
1. 산후우울증 → 짙은 안개 속 숲
2. 양육 번아웃 → 불 뿜는 용
3. 경력 단절 → 길 잃은 여우
4. 부부 갈등 → 갈라진 다리
5. 원가족 상처 → 가시 돋친 장미
6. 정체성 위기 → 거울 속 낯선 얼굴
7. 완벽주의 → 무거운 왕관
8. 외로움 → 빈 성
9. 죄책감 → 돌 짊어진 거인
10. 분노 → 화산
11. 불안 → 폭풍우
12. 상실 → 마른 강
13. 자존감 → 닳은 보석
14. 비교 → 그림자

## 전환 규칙

1. **전진만 가능** — Phase N → Phase N+1 (되돌릴 수 없음)
2. **최대 10턴** — 초과 시 자동 전환
3. **[PHASE:N] 태그** — AI 응답에서 추출
4. **서버사이드 검증** — 클라이언트 Phase보다 낮은 번호 거부

## 10장면 구조

```
[INTRO 1]      → 장면 1 (도입 1)
[INTRO 2]      → 장면 2 (도입 2)
[CONFLICT 1]   → 장면 3 (갈등 1)
[CONFLICT 2]   → 장면 4 (갈등 2)
[ATTEMPT 1]    → 장면 5 (시도 1)
[ATTEMPT 2]    → 장면 6 (시도 2)
[RESOLUTION 1] → 장면 7 (해결 1)
[RESOLUTION 2] → 장면 8 (해결 2)
[WISDOM 1]     → 장면 9 (교훈 1)
[WISDOM 2]     → 장면 10 (교훈 2)
```
