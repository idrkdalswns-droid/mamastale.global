# 동화 기반 AI 활동지 서비스, 아무도 안 한 블루오션을 설계하다

**마마스테일(MAMASTALE)이 구상하는 "AI 치유 동화 → 활동지 자동 생성" 서비스는 국내외 어디에도 존재하지 않는 완전한 시장 공백이다.** 한국과 글로벌 에듀테크 시장을 전수 조사한 결과, 동화 텍스트를 입력받아 누리과정 연계 활동지를 자동 생성하는 서비스는 단 하나도 없다. 키드키즈(130만 회원)와 꼬망세가 양분하는 한국 유아교육 콘텐츠 시장은 모두 **사람이 만든 정적 콘텐츠 라이브러리**이며, 해외 AI 워크시트 도구(Monsha, MagicSchool 등)는 텍스트 기반 문제 생성만 가능해 만3~5세에게 필요한 일러스트 중심 활동지는 생성하지 못한다. 동화 한 편에서 **72종 이상의 활동지**를 파생할 수 있고, 교사들은 활동지 제작에 하루 1~2시간을 소비하는 현실에서, 이 서비스는 수익성과 사용자 감동을 동시에 잡을 수 있는 강력한 포지션에 있다.

---

## 동화 한 편에서 72종 이상의 활동지가 나온다

동화 기반 활동지는 누리과정 5개 영역 전체를 커버하며, 단일 동화에서 파생 가능한 활동지 유형은 최소 **72종**이다. 이는 마마스테일 서비스의 "와, 이것까지?" 경험의 핵심 원천이 된다.

**의사소통 영역(14종 이상)** 이 가장 풍부하다. 글자따라쓰기(동화 속 핵심 어휘 활용), 동화 순서 맞추기(4~6컷 장면 배열), 스토리맵(시작-중간-끝 구조화), 독후활동지("내가 주인공이라면"), 편지쓰기(등장인물에게 편지), 빈칸 채우기, 낱말 퍼즐, 이야기 꾸미기/지어보기 등이 포함된다. **자연탐구 영역(15종 이상)**에서는 수세기(동화 속 동물/사물 세기), 패턴놀이(캐릭터 패턴 완성), 분류하기, 크기 비교, 도형 찾기, 그래프 만들기, 관찰 기록지, 생물 성장 순서 등이 가능하다.

**예술경험 영역(13종 이상)**에서는 색칠놀이(숫자별 색칠 포함), 점잇기, 오리기/붙이기, 역할놀이 대본, 캐릭터 가면/머리띠 만들기, 손인형 도안이 핵심이다. **사회관계 영역(8종 이상)**은 감정카드(장면별 주인공 기분 매칭), 감정 온도계, 관계도 그리기, 상황 판단 활동지 등이 있으며, 마마스테일의 **"치유 동화" 특성과 가장 시너지가 큰 영역**이다. **신체운동·건강 영역(8종 이상)**에서는 미로찾기, 선긋기, 가위질 연습, 건강습관/안전교육 활동지가 파생된다.

이 외에도 영역 통합 활동지(14종 이상)로 숨은그림찾기, 다른 그림 찾기, 그림자 찾기, 퍼즐, 보드게임, 빙고 등이 있다. 핵심 전략은 이 72종 전체를 한꺼번에 제공하는 것이 아니라, **동화 분석 결과에 따라 가장 적합한 8~12종을 자동 선별**하여 "활동 패키지"로 묶는 것이다.

| 누리과정 영역 | 활동지 유형 수 | 대표 활동지 |
|---|---|---|
| 의사소통 | 14+ | 글자따라쓰기, 순서맞추기, 스토리맵, 독후활동지 |
| 자연탐구 | 15+ | 수세기, 패턴놀이, 분류하기, 도형찾기 |
| 예술경험 | 13+ | 색칠놀이, 역할놀이 대본, 오리기/붙이기, 가면만들기 |
| 사회관계 | 8+ | 감정카드, 상황판단, 관계도, 약속만들기 |
| 신체운동·건강 | 8+ | 미로찾기, 선긋기, 가위질연습 |
| 영역 통합 | 14+ | 숨은그림찾기, 다른그림찾기, 보드게임 |

## 교사들의 진짜 고통: 매일 1~2시간을 활동지에 쓴다

한국 유아교육 교사의 페인 포인트를 정량적으로 분석한 결과, 활동지 서비스의 시장 기회는 명확하다. 육아정책연구소(KICCE) 연구에 따르면 **교사의 84.4%가 교재교구를 직접 제작**하며, 가장 큰 어려움으로 **"시간 부족"**을 꼽았다. 교사 1인이 하루 30분 단위로 평균 **1.41개의 업무를 동시 수행**하는 극단적 멀티태스킹 환경에서, 수업 준비 시간은 오후 3시 이후 아이들이 하원한 뒤에야 시작된다. 한 현직 교사의 증언에 따르면 "수업 하나를 구성하는 데 최소 한 시간이 걸린다."

현재 교사들의 활동지 제작 워크플로우는 극도로 비효율적이다. 한글(HWP)이나 파워포인트(PPT)로 직접 디자인하거나, 키드키즈·꼬망세에서 범용 활동지를 다운로드한 뒤 수정하거나, 핀터레스트·네이버 블로그에서 아이디어를 수집한 후 가위와 풀로 수작업한다. 특히 **동화 연계 활동은 플랫폼에서 제공하는 범용 월간/주간 활동지로는 해결되지 않아** 교사가 직접 연결고리를 만들어야 한다. 그림책사랑교사모임의 보고서는 이렇게 적시했다: "그림책 수업을 시작하는 선생님들은 그림책을 읽고 무엇을 해야 할지 몰라 막막해합니다."

교사 업무 부담의 구조적 문제도 심각하다. 한국교육개발원(KEDI) 2022년 데이터에 따르면 행정 업무 시간은 주당 **7.23시간**으로 2013년 대비 28% 증가했고, 실제 교육 시간은 18.72시간에서 **16.47시간으로 감소**했다. 교사 10명 중 6명이 행정 업무 때문에 수업 준비 시간이 부족하다고 응답했다. 이 환경에서 **"동화 한 편 → 활동지 패키지 60초 생성"**은 단순 편의를 넘어 교사의 삶의 질을 바꾸는 제안이 된다.

## 수익 모델: 티켓 + 구독 하이브리드가 최적해

한국 유아교육 콘텐츠 시장의 가격 구조와 글로벌 에듀테크 과금 모델을 교차 분석한 결과, 마마스테일에 최적화된 수익 모델은 **"프리미엄 + 티켓 소진 + 구독 옵션"의 3중 하이브리드 구조**다.

**한국 시장 벤치마크부터 보면**, 꼬망세 멀티회원이 연 ₩69,000(월 ₩5,750), 찬진교육 유아스토리가 연 약 ₩56,575(일 ₩155), 키드키즈가 티어별 유료 구독(정확한 가격 비공개)을 운영한다. 학부모 대상 프린트 학습지(시멘토, 참잘했어요)는 월 ₩5,000~31,200 수준이다. 해외는 Twinkl이 월 $5~12, Education.com이 월 $10~16, Teachers Pay Teachers가 팩당 $1~5다. 공통적으로 **무료 기본 접근 → 유료 전환**이 업계 표준이며, 한국 교사 시장은 **연간 결제 선호도**가 높다.

마마스테일의 활동지 원가는 놀라울 정도로 낮다. Claude API + DALL-E 3 기준 **활동지 10장(1패키지)당 약 $0.50~0.80**(약 ₩650~1,040), 자체 Stable Diffusion 호스팅 시 **$0.15~0.30**(약 ₩195~390)이다. 이 원가 구조에서 활동지 1장당 ₩200~400의 티켓 가격을 책정하면 **60~80%의 마진**을 확보할 수 있다.

| 티어 | 가격 | 내용 | 타겟 |
|---|---|---|---|
| 무료 체험 | ₩0 | 동화당 샘플 활동지 2~3장 | 전체 교사 (유입) |
| 체험팩 | ₩3,900 일회성 | 10티켓 (장당 ₩390) | 시범 사용자 |
| 월간이용권 | ₩6,900/월 | 30티켓/월 (장당 ₩230) | 정기 사용자 |
| 연간이용권 | ₩59,000/년 | 50티켓/월 (장당 ₩98) | 파워유저 |
| 기관 무제한 | ₩99,000~₩150,000/년 | 무제한 다운로드 | 유치원/어린이집 |

**수익 극대화 전략 5가지**: 첫째, 동화-활동지 콤보 패키지(AI 동화 + 활동지 10종 번들)를 단일 동화보다 30~50% 높은 티켓으로 책정한다. 둘째, 3월/9월 신학기 시즌팩을 월별 생활주제별로 큐레이션하여 한정 판매한다. 셋째, 개인 교사 → 기관 계정 업셀을 추진하여 ARPU를 5~10배 높인다. 넷째, 적립금/포인트 시스템(후기 작성, 추천 등)을 도입해 한국 시장의 문화적 기대에 부합한다. 다섯째, 수업계획안·가정통신문 자동 생성을 별도 프리미엄 기능으로 묶어 상위 티어 전환을 유도한다.

## "이런 것까지 해준다고?" 감탄 경험의 설계 공식

교사 감동을 이끌어내는 UX 포인트는 명확한 우선순위가 있다. 임팩트 순서대로 3개 티어로 구분하면 다음과 같다.

**티어 1(핵심 차별화, "이런 것까지?!")에 해당하는 3대 기능이 서비스의 운명을 결정한다.** 첫째, **동화 1편 → 다종 활동지 자동 생성**. 교사가 현재 하나의 활동지를 만드는 데 30~60분을 쓰는데, 8~12종의 활동지가 60초 만에 생성되는 경험은 "마법"으로 인식된다. 둘째, **누리과정 5개 영역 자동 매칭**. 각 활동지에 해당 누리과정 영역이 색상 코드 아이콘으로 표시되면, 교사가 평가/인정 문서 작성 시 즉시 활용할 수 있어 실무적 가치가 극대화된다. 셋째, **수업계획안 자동 생성**. 학습 목표, 시간 배분, 교사 발문, 예상 유아 반응, 평가 기준까지 포함된 완성형 수업계획안은 교사에게 주당 수 시간을 절약해 준다.

**티어 2(강력한 부가가치, "와, 이것도 돼?")**에는 연령별 자동 난이도 조정(만3세/4세/5세 혼합반 대응), 인쇄 즉시 가능한 A4 고품질 PDF, 원클릭 활동 패키지 다운로드(수업계획안 + 활동지 8종 + 가정통신문)가 해당한다. **티어 3(만족도 향상, "오, 편리하다!")**에는 수정 가능 템플릿(HWP 호환 필수), 월간/주간 주제 연동, 가정통신문 자동 생성, 교사 공유 커뮤니티가 있다.

UX 설계의 핵심 원칙은 **"3클릭 to 워크시트"**다. 동화 선택(서재에서 클릭 또는 전문 붙여넣기) → 연령/인원수 선택 → 활동지 패키지 미리보기 및 다운로드. 미리보기는 구매 전환에 결정적이므로, 생성된 모든 활동지의 **썸네일 갤러리를 반드시 제공**해야 한다. 출력 포맷은 PDF가 기본이되, HWP 호환은 한국 시장에서 필수적이다. 폰트는 교사들이 선호하는 무료 한글 폰트(마포꽃섬, 나눔고딕 등)를 사용하고, 일러스트 스타일은 한국 유아교육 현장의 따뜻한 손그림풍을 지향한다.

## AI 활동지 생성 에이전트의 기술 아키텍처

기술 파이프라인은 4단계로 설계한다. **Stage 1(텍스트 분석)**에서 Claude API 또는 GPT-4o가 동화 전문을 입력받아 등장인물, 주제, 핵심 어휘, 주요 장면, 감정, 수학/과학 개념, 교훈을 JSON 구조로 추출한다. **Stage 2(활동지 기획)**에서는 추출된 콘텐츠를 룰 엔진과 LLM이 결합하여 최적의 활동지 8~12종을 선정하고, 각 활동지의 텍스트 콘텐츠를 생성한다. **Stage 3(일러스트+템플릿)**에서 이미지 생성 AI가 캐릭터 일러스트/색칠도안/미로 등을 생성하고, Jinja2 템플릿 엔진이 HTML/CSS 워크시트 레이아웃에 콘텐츠를 배치한다. **Stage 4(PDF 어셈블리)**에서 WeasyPrint(Python) 또는 Puppeteer(Node.js)가 A4 인쇄 최적화 PDF를 최종 출력한다.

이미지 생성은 **MVP 단계에서 DALL-E 3 API(이미지당 $0.04~0.12)**를 사용하고, 프로덕션 단계에서 **Stable Diffusion XL + CoolKids v2 LoRA**(아동 일러스트 특화 모델)로 전환하는 것이 최적이다. 색칠놀이 도안은 "simple line art, coloring page, thick outlines, no shading" 프롬프트로 생성하거나, ColorBliss 같은 전문 도구의 API를 활용한다. **캐릭터 일관성**은 LoRA 파인튜닝(참조 이미지 4~10장, 소비자용 GPU에서 약 1시간)으로 확보하며, 동일 동화에서 파생된 모든 활동지에 동일한 캐릭터 스타일을 유지할 수 있다.

참고할 기존 도구와 학술 연구로는 Monsha.ai(커리큘럼 정렬 워크시트 생성, 연 $120), MagicSchool AI(빈칸 채우기/선다형 생성), LessonPlanLM(Nature 2025, RAG 기반 수업계획안 자동 생성), StoryMate(ArXiv 2025, GPT-4+RAG로 동화 기반 대화 생성)가 있다. 그러나 **이 중 어떤 도구도 동화→일러스트 활동지 변환을 수행하지 않는다**는 점이 핵심이다.

**예상 API 비용**은 동화 1편당 활동지 10장 기준 ₩650~1,040(DALL-E 3)이며, 자체 SD 호스팅 전환 시 ₩195~390으로 절감된다. 활동지 패키지 판매가가 ₩2,000~4,000이면 마진율 **60~80%**를 달성한다.

| 기술 구성요소 | MVP 추천 | 프로덕션 추천 |
|---|---|---|
| 텍스트 분석/콘텐츠 생성 | Claude API | Claude API + 파인튜닝 |
| 이미지 생성 | DALL-E 3 API | Stable Diffusion XL + CoolKids LoRA |
| 색칠도안 생성 | DALL-E 3 (line art 프롬프트) | SD + ControlNet |

## 경쟁사는 이 조합을 아직 못 만들었다

한국 유아교육 콘텐츠 시장은 **키드키즈/누리놀이(EK Corp) vs 꼬망세**의 양강 구도에 찬진교육이 신흥 도전자로 부상하는 구조다. 세 플랫폼 모두 수십만 건의 인간 제작 콘텐츠 라이브러리를 보유하고 있으며, 꼬망세만이 "꼬망봇"이라는 AI 기능을 도입했으나 그 범위는 알림장·가정통신문·평가기록부 등 **행정 문서 자동화에 한정**되어 있다. 콘텐츠 자체를 AI로 생성하는 플랫폼은 없다.

해외 시장에서도 공백은 동일하다. MagicSchool·Monsha·Worksheets.ai 같은 AI 워크시트 도구는 텍스트 기반 문제(선다형, 빈칸 채우기) 생성에 특화되어 있어 **만3~5세에게 필요한 일러스트 중심 활동지를 만들지 못한다**. Teachers Pay Teachers에서 "storybook companion worksheets"는 인기 카테고리이지만(팩당 $3~8) 교사가 수작업으로 제작한다. Twinkl만이 자체 동화(Twinkl Originals) + 연계 활동지를 함께 제공하지만, 이 역시 인간이 만든 정적 콘텐츠이며 영국 커리큘럼 기반이다.

**마마스테일의 차별화 매트릭스는 명확하다.** AI 동화 생성(✅, 경쟁사 ❌) + AI 활동지 생성(✅, 경쟁사 ❌ 또는 텍스트만) + 동화→활동지 자동 브릿지(✅, 전 세계 ❌) + 치유/SEL 통합(✅, 경쟁사 극소수) + 누리과정 자동 정렬(✅, 해외 ❌) + 일러스트 포함 시각적 결과물(✅, 해외 AI 도구 ❌). 이 6가지 요소의 조합이 곧 기술적 해자(moat)다. 경쟁사가 이를 복제하려면 AI 동화 엔진 + 이미지 생성 파이프라인 + 한국 누리과정 도메인 지식 + 유아 교육 디자인 전문성을 동시에 갖추어야 하며, 이는 단기간에 구현하기 어렵다.

가장 현실적인 위협은 **꼬망세의 AI 확장**이지만, 꼬망세의 현재 AI 전략은 행정 효율화에 집중되어 있어 콘텐츠 생성으로의 전환에는 조직적·기술적 관성이 작용한다. 글로벌 AI 플랫폼(OpenAI, Google)이 교육 특화 도구를 출시할 가능성은 있으나, 한국어 + 누리과정 + 유아 일러스트라는 극도로 로컬화된 요구사항을 충족하기까지는 상당한 시간이 걸린다.

## 결론: 실행을 위한 핵심 전략 프레임

이 리서치가 드러낸 가장 중요한 인사이트는 세 가지다.

첫째, **시장 공백이 실재한다.** "동화 → 활동지 자동 변환"은 국내외 에듀테크 어디에서도 구현되지 않은 완전한 화이트 스페이스다. 교사들이 가장 고통받는 지점(동화 읽고 나서 뭘 해야 하지?)과 정확히 일치하므로, PMF(Product-Market Fit) 가능성이 높다.

둘째, **기술적으로 즉시 실현 가능하다.** Claude API + DALL-E 3 + WeasyPrint 조합으로 MVP를 2~3주 내에 구축할 수 있으며, 패키지당 원가 ₩650~1,040으로 **60~80% 마진**의 수익 구조가 성립한다. 활동지 5종(색칠놀이, 글자따라쓰기, 순서맞추기, 수세기, 감정카드)으로 시작해 점진적으로 72종까지 확장하는 로드맵이 효율적이다.

셋째, **"치유 동화 + 활동지"의 결합이 카테고리를 창출한다.** 단순 편의 도구가 아니라, 아이의 감정을 다루는 치유 동화에서 감정카드·상황판단·관계도 등 **사회정서학습(SEL) 활동지**가 자연스럽게 파생되는 구조는 경쟁사가 모방하기 어려운 고유한 브랜드 정체성이 된다. 이 포지셔닝을 조기에 선점하면, "치유 동화 활동지"라는 카테고리 자체가 마마스테일의 이름과 동의어가 될 수 있다.
